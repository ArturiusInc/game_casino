## Доступные команды

| Command         | Description                |
| --------------- | -------------------------- |
| `npm install`   | установить все зависимости |
| `npm start`     | запуск в режиме разработки |
| `npm run build` | билд для прода             |

## Инструкция для запуска

### для разработки

`npm install`
`npm start`

### для продакшена

`npm install`
`npm run build`

после чего запустить index.html с помощью Live Server
